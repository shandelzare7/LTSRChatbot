- **用户**: TestUser、明轩
- **模型配置**:

## 测试 1: "你好" (简单问候)

### 内心独白输出
```
哦，浩然给我打招呼了。虽然只是一句简单的"你好"，不过还是让我有点意外，
毕竟我们之间还不算特别熟悉。这算是他主动想要联系我的意思吗？说不定只是
礼貌性问候而已。...
```

### 提取的结构化信号
- **emotion_tag**: 好奇
- **attitude**: 享受
- **momentum_delta**: 0.5 (↑ 参与意愿上升)
- **topic_appeal**: 7.0/10
- **selected_move_ids**: [1, 2, 5, 7]

### 生成候选 (5个)
1. Move_1: "你好，浩然。很高兴收到你的问候。最近有没有什么特别的事情发生呢？"
2. Move_2: "...希望我们可以在接下来的交流中更加了解彼此。"
3. Move_5: "...如果有想聊的话题，随时可以分享哦。"
4. Move_7: "...感觉你很有礼貌，主动打招呼总是一个好的开始呢。"
5. Free: "最近怎么样？希望一切都顺利。"

### 评判结果
- **Winner**: Move_7 (第4个)
- **理由**: 简单友好，既回应问候，又没有过多引导，符合角色的平和态度

### 最终回复
```
你好，浩然。很高兴收到你的问候。最近怎么样？希望一切都顺利。
```

```

- **emotion_tag**: 期待
- **attitude**: 主动配合
- **momentum_delta**: 0.5
- **topic_appeal**: 7.0/10

项目。虽然没有特别大的变化，但我倒是享受这种安静的工作状态。你问这个问题，
感觉你对我还是挺关心的，真让我感到温暖呢！你最近工作上有什么新进展吗？
```

## 关键观察

### ✅ 系统表现良好的方面

1. **内心独白质量高**
   - gpt-4o + temp=0.85 生成的独白具有真实的意识流特征
   - 包含矛盾、犹豫、联想等真实心理过程
   - 不像总结报告，更像真实的内心世界

2. **结构化提取准确**
   - 从独白中正确识别情绪标签（好奇、期待、犹豫）
   - 准确评估态度（享受、主动配合、好奇）
   - momentum_delta 在合理范围内 (0.5)

3. **生成多样性好**
   - 5个候选表现出不同的沟通风格和内容操作
   - Move 之间差异明显（有询问、有哲学、有自我暴露、有确认）
   - Free 路由也有合理的输出

4. **评判决策合理**
   - gpt-4o judge 能理解独白的细微差别
   - 评判理由跟独白内容形成闭环
   - 最终选择既符合角色心理，又保持对话自然度
