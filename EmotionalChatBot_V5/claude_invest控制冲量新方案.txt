## 完整方案：用 invest 控制「对话冲量衰减」替代直接调制 style

### 核心思路

1. **`invest` 不再直接加权 style 维度**（移除 `+ 0.10 * invest` 等项）
2. **`invest` 映射到冲量衰减参数** `sustain`、`half_life`、`floor`
3. **衰减曲线产出 `momentum`**（0~1），作为乘数调制 style 输出
4. **Knapp 阶段语义一致**：阶段越深 invest 越高 → 维持轮数更长、衰减更慢、底线更高

### 第一步：State 层添加字段

`app/state.py` 的 `AgentState` 中添加：

```python
# 对话冲量

conversation_momentum: float      # 当前冲量值 0.0~1.0
```

### 第三步：style.py 完整改造

这是核心。下面是完整的新 `style.py`：

```python
"""Style 节点：根据关系、情绪、信号和阶段语境计算 12 维风格参数和 2 个门控变量（纯计算，无 LLM）。"""
from __future__ import annotations

import math
from typing import Any, Callable, Dict

from utils.tracing import trace_if_enabled
from app.state import AgentState


def _clip01(x: float) -> float:
    return max(0.0, min(1.0, x))


def _avg2(a: float, b: float) -> float:
    return (a + b) / 2.0


def _safe_get(d: Dict[str, Any], *keys: str, default: float = 0.0) -> float:
    current = d
    for key in keys:
        if isinstance(current, dict):
            current = current.get(key, {})
        else:
            return default
    if isinstance(current, (int, float)):
        return float(current)
    return default


# Knapp stage baseline（invest 现在仅用于冲量衰减参数，不直接调制 style）
STAGE_PROFILE: Dict[int, Dict[str, float]] = {
    1:  {"invest": 0.15, "ctx": 0.10},
    2:  {"invest": 0.25, "ctx": 0.20},
    3:  {"invest": 0.45, "ctx": 0.40},
    4:  {"invest": 0.60, "ctx": 0.55},
    5:  {"invest": 0.75, "ctx": 0.70},
    6:  {"invest": 0.68, "ctx": 0.65},
    7:  {"invest": 0.55, "ctx": 0.55},
    8:  {"invest": 0.40, "ctx": 0.45},
    9:  {"invest": 0.25, "ctx": 0.30},
    10: {"invest": 0.10, "ctx": 0.15},
}


def _get_stage_index(stage: Any) -> int:
    if isinstance(stage, int):
        return max(1, min(10, stage))
    if isinstance(stage, str):
        stage_lower = stage.lower()
        stage_map = {
            "initiating": 1, "experimenting": 2, "intensifying": 3,
            "integrating": 4, "bonding": 5, "differentiating": 6,
            "circumscribing": 7, "stagnating": 8, "avoiding": 9,
            "terminating": 10,
        }
        if stage_lower in stage_map:
            return stage_map[stage_lower]
        try:
            return max(1, min(10, int(stage)))
        except Exception:
            pass
    return 1


# ─── 冲量衰减引擎 ───

def _invest_to_momentum_params(invest: float) -> Dict[str, float]:
    """
    将 invest (0.10~0.75) 映射到冲量衰减参数。
    
    invest 越高（关系越深）→ sustain 越长、half_life 越大、floor 越高
    即：亲密关系中人们能聊更久才开始"疲倦"。
    
    映射区间：
      invest 0.10 → sustain=2,  half_life=6,  floor=0.15
      invest 0.75 → sustain=8,  half_life=20, floor=0.45
    """
    # 归一化 invest 到 0~1（原始范围 0.10~0.75）
    t = _clip01((invest - 0.10) / 0.65)

    sustain   = 2.0 + 6.0 * t      # 2 ~ 8 轮
    half_life = 6.0 + 14.0 * t     # 6 ~ 20 轮
    floor     = 0.15 + 0.30 * t    # 0.15 ~ 0.45

    return {"sustain": sustain, "half_life": half_life, "floor": floor}


def _momentum_curve(turn: int, sustain: float, half_life: float, floor: float) -> float:
    """
    对话冲量衰减曲线。
    - 前 sustain 轮：1.0（全力投入）
    - 之后指数衰减到 floor
    """
    if turn <= sustain:
        return 1.0
    t_eff = turn - sustain
    decay = math.exp(-0.693 * t_eff / max(half_life, 1.0))  # ln(2) ≈ 0.693
    return floor + (1.0 - floor) * decay


def _compute_momentum(
    turn: int,
    invest: float,
    friendly: float = 0.0,
    low_effort: float = 0.0,
) -> float:
    """
    计算当前对话冲量。
    
    基础衰减由 invest 决定，用户信号可局部调制：
    - 用户高好感 → 冲量小幅回弹
    - 用户敷衍 → 冲量加速下降
    """
    params = _invest_to_momentum_params(invest)
    momentum = _momentum_curve(turn, **params)

    # 用户信号调制
    boost = 0.12 * friendly - 0.08 * low_effort
    momentum = _clip01(momentum + boost)

    # 用户敷衍时额外衰减
    if low_effort > 0.6:
        momentum *= 0.88

    return momentum


# ─── Style 节点工厂 ───

def create_style_node(llm_invoker: Any = None) -> Callable[[AgentState], dict]:
    """
    创建 Style 节点：纯计算，根据关系、情绪、信号和阶段语境计算风格参数。
    """

    @trace_if_enabled(
        name="Style",
        run_type="chain",
        tags=["node", "style", "computation"],
        metadata={"state_outputs": ["style", "llm_instructions"]},
    )
    def style_node(state: AgentState) -> dict:
        # ── A. 输入提取 ──
        relationship_state = state.get("relationship_state") or {}
        mood_state = state.get("mood_state") or {}
        detection_signals = state.get("detection_signals") or {}
        current_stage = state.get("current_stage") or "initiating"

        # 1) 6维关系
        closeness = _clip01(float(relationship_state.get("closeness", 0.5) or 0.5))
        trust     = _clip01(float(relationship_state.get("trust", 0.5) or 0.5))
        liking    = _clip01(float(relationship_state.get("liking", 0.5) or 0.5))
        respect   = _clip01(float(relationship_state.get("respect", 0.5) or 0.5))
        warmth    = _clip01(float(relationship_state.get("warmth", 0.5) or 0.5))
        power     = _clip01(float(relationship_state.get("power", 0.5) or 0.5))

        # 2) PAD（-1..1 → 0..1）
        P_raw = mood_state.get("pleasure", 0.0) or 0.0
        A_raw = mood_state.get("arousal", 0.0) or 0.0
        D_raw = mood_state.get("dominance", 0.0) or 0.0
        P = _clip01((P_raw + 1.0) / 2.0) if P_raw < 0 or P_raw > 1 else _clip01(P_raw)
        A = _clip01((A_raw + 1.0) / 2.0) if A_raw < 0 or A_raw > 1 else _clip01(A_raw)
        D = _clip01((D_raw + 1.0) / 2.0) if D_raw < 0 or D_raw > 1 else _clip01(D_raw)
        if P == 0.0 and A == 0.0 and D == 0.0:
            P = A = D = 0.5

        # 3) busy
        busy = _clip01(mood_state.get("busyness", 0.0) or 0.0)

        # 4) detection_signals
        composite  = detection_signals.get("composite") or {}
        trace      = detection_signals.get("trace") or {}
        instant_eff = detection_signals.get("instant_eff") or {}
        stage_ctx  = detection_signals.get("stage_ctx") or {}

        pos   = _safe_get(composite, "goodwill", default=0.0)
        neg   = _safe_get(composite, "conflict_eff", default=0.0)
        prov  = _safe_get(composite, "provocation", default=0.0)
        press = _safe_get(composite, "pressure", default=0.0)
        uncert = _safe_get(trace, "confusion", default=0.0)
        if uncert == 0.0:
            uncert = _safe_get(instant_eff, "confusion", default=0.0)

        # 5) stage_ctx
        too_close_too_fast      = _safe_get(stage_ctx, "too_close_too_fast", default=0.0)
        too_distant_too_cold    = _safe_get(stage_ctx, "too_distant_too_cold", default=0.0)
        betrayal_violation      = _safe_get(stage_ctx, "betrayal_violation", default=0.0)
        over_caring             = _safe_get(stage_ctx, "over_caring", default=0.0)
        dependency_bid          = _safe_get(stage_ctx, "dependency_bid", default=0.0)
        possessiveness_jealousy = _safe_get(stage_ctx, "possessiveness_jealousy", default=0.0)
        power_move              = _safe_get(stage_ctx, "power_move", default=0.0)
        stonewalling_intent     = _safe_get(stage_ctx, "stonewalling_intent", default=0.0)

        # 6) Knapp stage
        stage_index = _get_stage_index(current_stage)

        # ── B. 派生量 ──
        Aff  = _clip01(0.55 * liking + 0.25 * warmth + 0.20 * closeness)
        Saf  = _clip01(0.50 * trust + 0.35 * respect + 0.15 * closeness)
        PowC = power - 0.50

        stage_profile = STAGE_PROFILE.get(stage_index, STAGE_PROFILE[1])
        invest = stage_profile["invest"]
        ctx    = stage_profile["ctx"]
        break_n = 0.0 if stage_index <= 5 else (stage_index - 5) / 5.0

        BoundaryNeed = _clip01(
            0.45 * betrayal_violation + 0.35 * power_move +
            0.25 * stonewalling_intent + 0.20 * too_distant_too_cold +
            0.20 * possessiveness_jealousy + 0.15 * over_caring
        )
        Unease = _clip01(
            0.35 * too_close_too_fast + 0.25 * dependency_bid +
            0.25 * over_caring + 0.20 * possessiveness_jealousy +
            0.15 * power_move
        )

        # ── C. 计算冲量 ──
        turn = state.get("turn_count_in_session") or 1

        # 从 detection_signals 获取 friendly / low_effort 用于冲量调制
        scores = detection_signals.get("scores") or {}
        friendly   = float(scores.get("friendly", 0.0) or 0.0)
        low_effort = float(scores.get("low_effort", 0.0) or 0.0)

        momentum = _compute_momentum(
            turn=turn,
            invest=invest,
            friendly=friendly,
            low_effort=low_effort,
        )

        # ── D. 12维 style 公式（已移除 invest 的直接调制项）──

        # 1) self_disclosure
        self_disclosure = _clip01(
            0.10
            + 0.55 * _avg2(trust, closeness)
            - 0.25 * A
            + 0.15 * pos
            # 原: + 0.10 * invest → 已移除
        )

        # 2) topic_adherence
        topic_adherence = _clip01(
            0.20
            + 0.70 * respect
            - 0.25 * uncert
            - 0.15 * prov
            + 0.08 * (1 - ctx)
        )

        # 3) initiative
        initiative = _clip01(
            0.15
            + 0.45 * power
            + 0.35 * liking
            - 0.35 * busy
            + 0.10 * neg
            # 原: + 0.08 * invest → 已移除
        )

        # 4) advice_style
        advice_style = _clip01(
            0.10
            + 0.45 * power
            + 0.35 * liking
            + 0.20 * BoundaryNeed
            - 0.20 * busy
            # 原: + 0.06 * invest → 已移除
        )

        # 5) subjectivity（原本无 invest 项，不变）
        subjectivity = _clip01(
            0.35
            + 0.55 * power
            - 0.45 * respect
            + 0.30 * D
            + 0.25 * BoundaryNeed
            + 0.10 * break_n
        )

        # 6) memory_hook（原本无 invest 项，不变）
        memory_hook = _clip01(
            0.05
            + 0.80 * closeness
            + 0.10 * pos
            - 0.25 * busy
            + 0.15 * ctx
        )

        # 7) verbal_length
        verbal_length = _clip01(
            0.20
            + 0.45 * _avg2(warmth, closeness)
            - 0.55 * busy
            - 0.20 * neg
            - 0.15 * BoundaryNeed
            # 原: + 0.10 * invest → 已移除
            - 0.20 * break_n
        )

        # 8) social_distance（原本无 invest 项，不变）
        social_distance = _clip01(
            0.40
            + 0.40 * power
            + 0.25 * respect
            - 0.55 * closeness
            + 0.20 * neg
            + 0.25 * BoundaryNeed
            - 0.20 * ctx
            + 0.30 * break_n
        )

        # 9) tone_temperature
        tone_temperature = _clip01(
            0.20
            + 0.45 * _avg2(warmth, liking)
            + 0.25 * P
            - 0.25 * neg
            - 0.20 * Unease
            # 原: + 0.10 * invest → 已移除
            - 0.25 * break_n
        )

        # 10) emotional_display
        emotional_display = _clip01(
            0.10
            + 0.45 * _avg2(trust, closeness)
            + 0.35 * A
            - 0.25 * social_distance
            # 原: + 0.08 * invest → 已移除
            - 0.12 * break_n
        )

        # 11) wit_and_humor（原本无 invest 项，不变）
        wit_and_humor = _clip01(
            0.05
            + 0.45 * _avg2(liking, closeness)
            + 0.20 * pos
            - 0.30 * BoundaryNeed
            - 0.20 * neg
            + 0.15 * ctx
            - 0.25 * break_n
        )

        # 12) non_verbal_cues（原本无 invest 项，不变）
        non_verbal_cues = _clip01(
            0.05
            + 0.65 * closeness
            + 0.10 * pos
            - 0.45 * busy
            - 0.20 * social_distance
            + 0.10 * ctx
            - 0.15 * break_n
        )

        # ── E. 冲量调制：对受影响的维度施加衰减乘数 ──
        # momentum=1.0 时无影响；momentum→floor 时抑制最大
        # m_factor 将 momentum 映射为一个温和的乘数（不会把值压到 0）
        m_soft  = 0.55 + 0.45 * momentum   # 0.62 ~ 1.0（较温和）
        m_steep = 0.40 + 0.60 * momentum   # 0.49 ~ 1.0（较强）

        self_disclosure    = _clip01(self_disclosure * m_soft)
        initiative         = _clip01(initiative * m_steep)      # 主动性衰减明显
        verbal_length      = _clip01(verbal_length * m_steep)   # 话多→话少
        tone_temperature   = _clip01(tone_temperature * m_soft)
        emotional_display  = _clip01(emotional_display * m_soft)
        wit_and_humor      = _clip01(wit_and_humor * m_soft)
        non_verbal_cues    = _clip01(non_verbal_cues * m_soft)

        # ── F. 两个门控变量 ──
        coldness_gate = _clip01(
            0.10
            + 0.45 * stonewalling_intent
            + 0.25 * too_distant_too_cold
            + 0.25 * neg
            + 0.25 * busy
            - 0.20 * _avg2(closeness, warmth)
            + 0.35 * break_n
            - 0.10 * ctx
        )

        boundary_gate = _clip01(
            0.10
            + 0.45 * betrayal_violation
            + 0.25 * power_move
            + 0.20 * press
            + 0.15 * prov
            + 0.10 * D
            - 0.20 * Saf
            + 0.20 * break_n
            # 原: + 0.10 * invest → 已移除
        )

        # ── G. mode.style_bias ──
        mode = state.get("current_mode")
        if mode and hasattr(mode, "style_bias"):
            bias = mode.style_bias
            if hasattr(bias, "verbal_length") and bias.verbal_length is not None:
                verbal_length = _clip01(0.7 * verbal_length + 0.3 * bias.verbal_length)
            if hasattr(bias, "tone_temperature") and bias.tone_temperature is not None:
                tone_temperature = _clip01(0.7 * tone_temperature + 0.3 * bias.tone_temperature)
            if hasattr(bias, "social_distance") and bias.social_distance is not None:
                social_distance = _clip01(0.7 * social_distance + 0.3 * bias.social_distance)
            if hasattr(bias, "advice_style") and bias.advice_style is not None:
                advice_style = _clip01(0.7 * advice_style + 0.3 * bias.advice_style)
            if hasattr(bias, "wit_and_humor") and bias.wit_and_humor is not None:
                wit_and_humor = _clip01(0.7 * wit_and_humor + 0.3 * bias.wit_and_humor)
            if hasattr(bias, "emotional_display") and bias.emotional_display is not None:
                emotional_display = _clip01(0.7 * emotional_display + 0.3 * bias.emotional_display)

        # ── H. 输出 ──
        style_output = {
            "self_disclosure": self_disclosure,
            "topic_adherence": topic_adherence,
            "initiative": initiative,
            "advice_style": advice_style,
            "subjectivity": subjectivity,
            "memory_hook": memory_hook,
            "verbal_length": verbal_length,
            "social_distance": social_distance,
            "tone_temperature": tone_temperature,
            "emotional_display": emotional_display,
            "wit_and_humor": wit_and_humor,
            "non_verbal_cues": non_verbal_cues,
            "coldness_gate": coldness_gate,
            "boundary_gate": boundary_gate,
            "conversation_momentum": round(momentum, 3),
            "derived": {
                "Aff": Aff,
                "Saf": Saf,
                "BoundaryNeed": BoundaryNeed,
                "Unease": Unease,
                "invest": invest,
                "ctx": ctx,
                "break_n": break_n,
                "momentum_params": _invest_to_momentum_params(invest),
                "turn": turn,
            },
        }

        llm_instructions = {
            "self_disclosure": self_disclosure,
            "topic_adherence": topic_adherence,
            "initiative": initiative,
            "advice_style": advice_style,
            "subjectivity": subjectivity,
            "memory_hook": memory_hook,
            "verbal_length": verbal_length,
            "social_distance": social_distance,
            "tone_temperature": tone_temperature,
            "emotional_display": emotional_display,
            "wit_and_humor": wit_and_humor,
            "non_verbal_cues": non_verbal_cues,
        }

        print(
            f"[Style] 12D computed: disclosure={self_disclosure:.2f}, "
            f"length={verbal_length:.2f}, temp={tone_temperature:.2f}, "
            f"momentum={momentum:.2f} (turn={turn}), "
            f"gates: cold={coldness_gate:.2f}, boundary={boundary_gate:.2f}"
        )

        return {
            "style": style_output,
            "llm_instructions": llm_instructions,
            "conversation_momentum": round(momentum, 3),
            "style_analysis": f"12D computed (momentum={momentum:.2f}, cold_gate={coldness_gate:.2f}, boundary_gate={boundary_gate:.2f})",
        }

    return style_node
```

---

### 变更总结

| 项目 | 旧做法 | 新做法 |
|---|---|---|
| **`invest` 在 style 公式中** | 直接 `+ W * invest`（6个维度 + boundary_gate） | **全部移除** |
| **`invest` 的新用途** | — | `_invest_to_momentum_params()` → 控制 `sustain/half_life/floor` |
| **冲量计算** | 不存在 | `_momentum_curve()` + `_compute_momentum()`（含用户信号调制） |
| **冲量对 style 的作用** | — | 衰减乘数 `m_soft`/`m_steep` 作用于 7 个维度 |
| **State 新增** | — | `turn_count_in_session`, `conversation_momentum` |
| **Loader 改动** | — | 每轮递增 `turn_count_in_session` |

| 4 integrating | 0.60 | 6.5 | 16.8 | 0.38 | 融合，长聊不倦 |
| 5 bonding | 0.75 | 8.0 | 20.0 | 0.45 | 深度绑定，最持久 |
| 6 differentiating | 0.68 | 7.3 | 18.5 | 0.42 | 开始分化但仍有惯性 |
| 7 circumscribing | 0.55 | 6.1 | 15.4 | 0.36 | 话题收窄 |

| 10 terminating | 0.10 | 2.0 | 6.0 | 0.15 | 快结束了，冲量最低 |

### 冲量衰减曲线示例（bonding 阶段，invest=0.75）

```
Turn  1: momentum=1.00  →  m_steep=1.00, m_soft=1.00
Turn  5: momentum=1.00  →  (sustain 期)
Turn  8: momentum=1.00  →  (刚过 sustain=8)
Turn 10: momentum=0.93  →  m_steep=0.96, m_soft=0.97
Turn 15: momentum=0.79  →  m_steep=0.87, m_soft=0.91
Turn 20: momentum=0.67  →  m_steep=0.80, m_soft=0.85
Turn 30: momentum=0.53  →  m_steep=0.72, m_soft=0.79
Turn 50: momentum=0.46  →  m_steep=0.68, m_soft=0.76
```

对比 initiating 阶段 (invest=0.15)：
```
Turn  1: momentum=1.00
Turn  3: momentum=0.93  →  (sustain=2.5 已过)
Turn  8: momentum=0.59
Turn 15: momentum=0.30
Turn 20: momentum=0.21  →  接近 floor=0.17
```

这完美体现了"阶段越深，聊得越久才衰减"的语义。

### 下游消费点提示

`conversation_momentum` 已写入 state，下游节点（task_planner、detection、processor）可直接 `state.get("conversation_momentum", 1.0)` 获取，用于：
- **task_planner**: 冲量低时降低 `word_budget` 建议
- **detection**: 冲量低时减少好奇驱动的 `immediate_tasks`
- **processor**: 冲量低时降低 `fragmentation_tendency`

这些下游改动属于增量优化，可后续按需添加，核心机制已在 style 节点中闭环。
